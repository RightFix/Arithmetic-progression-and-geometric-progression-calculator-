# Arithmetic & Geometric Progression Calculator

A modern, user-friendly web calculator for solving Arithmetic Progression (AP) and Geometric Progression (GP) problems. Perfect for students and anyone learning about sequences and series.

## What is AP & GP?

**Arithmetic Progression (AP)** - A sequence where each term differs from the previous by a constant value (common difference). Example: 2, 5, 8, 11...

**Geometric Progression (GP)** - A sequence where each term is found by multiplying the previous term by a constant (common ratio). Example: 2, 6, 18, 54...

## Features

### Arithmetic Progression
- Find First Term (given nth term, second term, sum & last term, or sum & common difference)
- Find Second Term
- Find Common Difference (given second term, nth term, or sum of terms)
- Find Nth Term
- Find Number of Terms (given nth term, sum & last term, or sum & common difference)
- Find Last Term (given common difference or sum)
- Find Sum of Terms (given last term or common difference)

### Geometric Progression
- Find First Term (given second term, nth term, sum r>1, sum r<1, or infinite sum)
- Find Second Term
- Find Common Ratio (given second term, nth term, or infinite sum)
- Find Nth Term
- Find Number of Terms (given nth term, r<1, or r>1)
- Find Sum of Terms (for r>1 and r<1)
- Find Sum of Infinite Terms

### Additional Features
- Built-in formulas and explanations for AP and GP
- Educational reference section with examples
- Responsive design - works on mobile and desktop

## How to Use

1. Open `index.html` in any web browser
2. Select AP or GP section
3. Enter the values you know
4. Click the appropriate solve button
5. Use the Reset button to clear all fields
