# Arithmetic & Geometric Progression Calculator

A web-based calculator for solving Arithmetic Progression (AP) and Geometric Progression (GP) problems.

## Features

### Arithmetic Progression
- Find First Term (4 methods)
- Find Second Term
- Find Common Difference (3 methods)
- Find Nth Term
- Find Number of Terms (3 methods)
- Find Last Term (2 methods)
- Find Sum of Terms (2 methods)

### Geometric Progression
- Find First Term (5 methods)
- Find Second Term
- Find Common Ratio (3 methods)
- Find Nth Term
- Find Number of Terms (3 methods)
- Find Sum of Terms (2 methods)
- Find Sum of Infinite Terms

## Usage

Open `index.html` in any web browser. No server required.

## Files

- `index.html` - Main HTML file
- `progression.js` - JavaScript calculations
- `APGP.css` - Styling
- `logo.jpg` - Project logo
